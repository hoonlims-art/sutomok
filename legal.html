개발행위 법령 검토

사업지 주소와 용도를 입력하면 AI가 관련 법령·지침을 실시간으로 검색·분석하여 개발행위 가능 여부를 검토해 드립니다.

🔑 API 키 발급 방법 (처음 사용 시 1회만)

법령 검토 AI 기능을 사용하려면 Anthropic API 키가 필요합니다. 아래 순서대로 진행해 주세요.

1
Anthropic Console 접속
https://console.anthropic.com 으로 접속합니다.
구글 계정 또는 이메일로 무료 회원가입을 진행합니다.
2
결제 수단 등록
왼쪽 메뉴 → Billing → Add credit card
신용카드를 등록합니다. 최소 $5(약 7,000원) 충전 후 사용량만큼 차감됩니다.
검토 1회 비용은 약 5~20원 수준으로 매우 저렴합니다.
3
API 키 생성
왼쪽 메뉴 → API Keys → Create Key 클릭
키 이름을 입력(예: 수토목법령검토)하고 Create Key 버튼을 누릅니다.
생성된 키(sk-ant-api03-...)를 복사합니다.
⚠️ 이 키는 한 번만 표시되므로 반드시 복사해 두세요.
4
아래 입력창에 키 붙여넣기 후 저장
복사한 키를 아래 API 키 입력창에 붙여넣고 저장 버튼을 클릭합니다.
저장된 키는 이 브라우저에 보관되어 다음 방문 시 다시 입력하지 않아도 됩니다.
💰 예상 비용 안내
최소 충전액: $5 (약 7,000원) · 검토 1회 비용: 약 5~20원 수준
$5 충전 시 약 250~1,000회 검토 가능 · 사용하지 않으면 잔액 유지
🔑 Anthropic API 키 입력 미입력
입력한 키는 이 브라우저에만 저장되며 외부로 전송되지 않습니다.
